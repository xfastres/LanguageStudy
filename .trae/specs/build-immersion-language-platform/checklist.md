# Checklist

## 用户注册登录系统
- [ ] 用户可通过邮箱+密码成功注册，注册后自动登录并获取JWT令牌
- [ ] 用户可通过邮箱+密码成功登录，获取JWT令牌并跳转学习主页
- [ ] 重复邮箱注册时返回明确的错误提示
- [ ] JWT认证中间件正确保护需登录的路由，未登录请求返回401
- [ ] 前端认证状态管理正确：未登录自动重定向至登录页，token过期自动处理

## 三阶段分级课程体系
- [ ] 课程数据模型支持三阶段分级、多语种、难度标签
- [ ] 课程API支持按语言、阶段、难度查询课程列表和详情
- [ ] 内容难度分级引擎能基于多维度计算难度分数
- [ ] 课程首页正确展示语言选择、三阶段课程卡片
- [ ] 阶段说明页完整展示各阶段核心目标、执行动作、绝对禁止、达标标准

## 听力训练模块（核心模块）
- [ ] 听力内容API能按阶段、难度、主题查询音频/视频内容
- [ ] 听力播放器支持逐段播放、重复播放、播放速度调节
- [ ] 听懂率自评组件在每段内容播放后弹出，80%以上才计入有效输入时长
- [ ] 有效输入时长正确记录并累计
- [ ] 字幕控制遵循阶段规则：阶段一禁用双语字幕，阶段二优先单字幕，阶段三减少依赖

## 单词记忆模块
- [ ] 词汇提取逻辑能从用户已听内容中提取高频词汇并关联上下文
- [ ] 单词卡片展示词汇在听力内容中的原句上下文，非孤立背诵
- [ ] 词汇感知练习通过上下文猜测词义，非默写测试

## 语法练习模块
- [ ] 语法结构提取逻辑能从用户已听内容中提取语法结构并关联例句
- [ ] 语法感知页面展示大量例句让用户自然感知规律，非规则背诵
- [ ] 语法情境练习在可理解输入中识别语法结构，非填空改错

## 口语跟读模块
- [ ] 阶段二提供可选跟读内容，阶段三全面开放
- [ ] 跟读播放器支持播放原音→录制跟读→回放对比
- [ ] 语音识别提供提示性反馈，非纠错性反馈
- [ ] 阶段一用户访问口语跟读时显示温和提示，不强制阻止

## 学习进度追踪系统
- [ ] 有效输入时长作为核心指标正确记录和展示
- [ ] 进度仪表盘展示：有效输入时长、阶段进度环、连续天数streak
- [ ] 学习时长统计图表正确展示每日/每周趋势
- [ ] 阶段晋级判定逻辑正确：累计有效输入时长达标+评估通过
- [ ] 学习连续性追踪正确：streak计算、断连重置

## 个性化学习路径推荐
- [ ] 初始水平评估测试能确定用户初始阶段和起点
- [ ] i+1难度推荐引擎基于用户水平+兴趣+听懂率计算推荐难度区间
- [ ] 学习路径规划API生成从当前水平到目标水平的完整路径
- [ ] 兴趣偏好设置影响推荐内容
- [ ] 动态难度调整：听懂率>90%推荐更高难度，<80%推荐更低难度

## 社区交流系统
- [ ] 帖子CRUD功能完整：发布、查询、点赞、评论
- [ ] 评论内容过滤纠错类内容，遵循零焦虑原则
- [ ] 语伴匹配仅对阶段三用户开放，基于目标语言和兴趣匹配
- [ ] 学习小组支持创建、加入、打卡、成员进度展示

## 成就激励系统
- [ ] 里程碑成就（50h/100h/300h/600h/1000h）在有效输入时长达标时自动颁发
- [ ] 连续学习成就（7天/30天/100天）在连续天数达标时自动颁发
- [ ] 阶段晋级成就晋级时自动颁发
- [ ] 成就展示页面完整：徽章列表、进度、成长时间线
- [ ] 获得新成就时弹出庆祝动画

## 核心总则约束
- [ ] 听力训练模块在首页和学习路径中占据最核心位置，入口优先级最高
- [ ] 全平台无强制计时、无错误率排名、无强制完成量、无与他人直接对比的竞争性数据
- [ ] 阶段一用户尝试使用口语跟读或语伴功能时显示温和提示

## 学习主页与导航整合
- [ ] 学习主页听力训练入口优先，展示今日推荐、进度概览、连续天数
- [ ] 全局导航完整：学习、课程、进度、社区、成就、个人中心
- [ ] 个人中心包含个人信息、目标语言设置、兴趣偏好、学习统计
- [ ] 响应式布局适配桌面和移动端

## 种子数据与端到端验证
- [ ] 种子数据脚本包含英语/日语/韩语三阶段示范课程和听力内容元数据
- [ ] 端到端流程可走通：注册→评估→课程→听力训练→进度追踪→阶段晋级→社区→成就
