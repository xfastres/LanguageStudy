# Checklist

## 用户注册登录与初始评估
- [ ] 用户可通过邮箱+密码成功注册，注册后自动登录并初始化UserAcquisitionProfile
- [ ] 用户可通过邮箱+密码和Google OAuth登录，登录后跳转输入流首页
- [ ] 重复邮箱注册返回明确错误提示
- [ ] JWT认证守卫正确保护需登录路由，未登录返回401
- [ ] 初始评估流程完整：目标语言选择→兴趣偏好→沉浸式水平评估→确定初始输入区间
- [ ] 零基础用户看到欢迎引导："最重要的就是多听，不需要背单词、不需要开口"

## 持续可理解输入流（Input Feed）
- [ ] 输入流首页展示个性化i+1内容流，支持无限下滑浏览
- [ ] 内容卡片展示缩略图/波形、时长、主题标签，强调内容吸引力而非"学习属性"
- [ ] 输入流根据用户实时行为信号动态调整下一条推荐难度
- [ ] 兴趣主题筛选正确影响推荐内容
- [ ] "继续沉浸"入口正确恢复上次沉浸位置

## 可理解性评分引擎（平台灵魂）
- [ ] UserAcquisitionProfile所有维度正确读写和持续更新
- [ ] ContentFeatures所有维度正确提取和存储
- [ ] comprehensionScore计算模型正确输出0-1分数
- [ ] i+1区间判定正确：0.8-0.92为i+1，<0.8太难，>0.92太简单
- [ ] 评分动态修正：根据用户实际行为信号修正评分并更新画像
- [ ] 用户行为信号采集API正确接收和存储所有信号类型

## 沉浸式输入播放器（平台最重要页面）
- [ ] 播放器核心功能完整：全屏、播放/暂停、进度拖拽、音量控制
- [ ] 播放速度调节0.5x-2.0x，降速不变调
- [ ] AB循环播放正确工作
- [ ] 自动句块切分正确，支持逐句前进/后退/重复
- [ ] 智能字幕系统：目标语言字幕、母语翻译可选、透明度渐隐滑块、点击词弹出上下文释义
- [ ] 字幕渐隐机制：根据字幕查看频率自动建议降低透明度
- [ ] 仅音频模式正确隐藏视频保留音频+字幕
- [ ] 视觉辅助强度调节正确工作（高辅助→低辅助）
- [ ] 跟读影子模式：每句后留跟读间隔，语音识别提示性反馈，仅"日常场景输入"及以上区间开放
- [ ] "帮我理解"AI解释功能：生成简化目标语言解释而非母语翻译
- [ ] 沉浸输入完成反馈面板：展示有效输入时长+自然接触的表达，自动过渡下一条
- [ ] 行为信号实时采集上报正确

## AI内容加工流水线
- [ ] Whisper自动转录与分句正确，生成时间轴对齐字幕
- [ ] 难度分析与特征提取正确计算所有维度
- [ ] 关键词与表达提取正确关联上下文例句
- [ ] LLM多语言字幕生成自然流畅
- [ ] 慢速版生成降速不变调
- [ ] 简化版生成（LLM简化文本+TTS音频）正确
- [ ] 自动打标正确：场景、兴趣、视觉辅助评分、适配区间
- [ ] 内容入库API触发AI Pipeline异步处理，状态回调正确

## 多信号有效输入时长计算
- [ ] 有效输入时长计算模型综合多信号（非单一自评），输出effectiveMinutes
- [ ] 高字幕依赖+高频暂停+低完成度=低有效系数；低字幕依赖+适度回放+高完成度=高有效系数
- [ ] 快速理解检查组件轻量级弹出，非测试性质
- [ ] 有效输入时长展示：累计、今日、连续沉浸天数

## 语言发现与表达感知
- [ ] 沉浸输入完成后表达自然浮现，展示"你已经自然接触了"列表，每条关联上下文例句
- [ ] 不使用"今日单词""语法练习"等触发应试模式的命名
- [ ] 表达深度探索展示所有出现实例、上下文场景、用法归纳（归纳式非规则式）
- [ ] 语法结构自然感知展示多组例句，不提供"语法规则说明"
- [ ] 表达收藏功能正确，每条可回溯到原始听力场景
- [ ] 所有知识依附于输入内容存在，绝不独立出现

## 隐形阶段系统
- [ ] 后台三阶段判定逻辑正确（基于有效输入时长+可理解性评分+行为信号）
- [ ] 前台展示为中性"输入区间"：慢速视觉辅助输入/日常场景输入/自然母语输入
- [ ] 不使用"阶段""等级""初级/中级/高级"等比较性词汇
- [ ] 区间过渡提示中性："你的输入世界正在扩展"，而非"恭喜晋级"
- [ ] 功能按区间自然开放：慢速→仅输入+发现；日常→+跟读+AI陪伴；自然→+AI对话+语伴
- [ ] 低区间用户尝试高区间功能时温和提示"多听就好"，不强制阻止

## AI沉浸式陪伴教师
- [ ] AI对话用目标语言，语速和词汇适配用户水平，不纠错，只正向引导
- [ ] AI表达解释用简化目标语言而非母语翻译
- [ ] AI推荐解释用目标语言说明理由
- [ ] AI鼓励消息强调"你已经在自然习得"而非"你完成了任务"
- [ ] "慢速视觉辅助输入"区间用户尝试AI对话时温和提示，不强制阻止
- [ ] WebSocket实时对话正常工作

## 陪伴型社区
- [ ] 发布格式引导为输入体验分享，非能力展示
- [ ] 互动方式仅限共鸣/鼓励/推荐，禁止纠错类和能力比较类评论
- [ ] 沉浸伙伴匹配仅对"自然母语输入"区间用户开放
- [ ] 沉浸小组聚焦共同沉浸打卡、主题输入挑战、互相推荐内容

## 成就激励系统
- [ ] 沉浸里程碑（50h/100h/300h/600h/1000h）自动颁发，展示里程碑意义
- [ ] 连续沉浸成就（7/30/100天）自动颁发，强调"习惯的力量"
- [ ] 输入世界扩展成就区间过渡时自动颁发
- [ ] 成就文案强调"自然习得"而非"努力学习"
- [ ] 成就弹窗通知含庆祝动画+里程碑意义说明

## 核心总则约束
- [ ] 输入流和沉浸式播放器占据首页和导航最核心位置
- [ ] 全平台无强制计时、无错误率排名、无强制完成量、无与他人直接对比数据
- [ ] 不使用"阶段""等级""初级/中级/高级"等比较性词汇
- [ ] 所有语言知识依附于输入内容存在，绝不独立呈现
- [ ] 低区间用户尝试高区间功能时温和提示而非强制阻止

## 沉浸主页、导航与个人中心
- [ ] 沉浸主页以输入流为核心入口，展示今日精选、有效输入时长、连续天数
- [ ] 全局导航"沉浸"入口最突出
- [ ] 个人中心包含目标语言设置、兴趣偏好调整、输入区间展示
- [ ] 进度页面以有效输入时长为核心指标
- [ ] 响应式布局适配桌面和移动端
